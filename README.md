# README

## System dependencies

- PostgreSQL
- Ruby version 2.4.0
- Rails version 5.1.4

## Setup

Clone repository:

    https://github.com/pvtmike123/Electronic-Health-Record.git

DB creation:

    rake db:create

DB migration:

    rake db:migrate

DB seed:

    rake db:seed

## Running

Run locally:

    rails s
