# README

## System dependencies

- PostgreSQL
- Ruby version 2.4.0
- Rails version 5.1.4

## Setup

Clone repository:

    https://github.com/pvtmike123/Electronic-Health-Record.git

Then:

    `bundle install`

## Database creation

Create:

    rake db:create

Migrate:

    rake db:migrate

Seed:

    rake db:seed

## Run

    rails s
